
# AI Attributions

This project was developed by the team with the support of various AI-assisted tools used for research, ideation, debugging, documentation, and productivity enhancement. All outputs generated by AI tools were reviewed, modified, validated, and integrated manually by team members before inclusion in the final system.

## AI Tools and Usage

### ChatGPT (OpenAI)

Used for:

* Generating and refining synthetic civic complaint datasets for model training
* Providing guidance on GitHub workflows, deployment, and project organization

### Claude (Anthropic)

Used for:

* System architecture brainstorming
* Architecture diagram generation and refinement
* High-level design discussions and component planning
* Assisting with documentation, README preparation, and technical writing

### Gemini (Google)

Used for:

* Debugging assistance
* Error analysis and troubleshooting
* Development workflow suggestions
* Validation of implementation approaches

### AI Code Completion Tools

Used where applicable for:

* Code suggestions
* Productivity improvements during development

## Human Review and Validation

All AI-generated suggestions, code snippets, diagrams, datasets, documentation drafts, and implementation ideas were manually reviewed, modified, tested, and validated by the team.

The final decisions regarding:

* Project architecture
* Dataset design
* Machine learning model implementation
* Backend services
* Frontend interfaces
* Testing procedures
* Deployment strategy

were made entirely by the team.

## Final Responsibility

The team retains full responsibility for the design, implementation, testing, validation, and correctness of the submitted project. AI tools were used solely as development aids and productivity enhancers and did not replace human decision-making during project development.
